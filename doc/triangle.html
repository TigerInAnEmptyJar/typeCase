    • void triangle::Draw(const point3D& eye = point3D(0,0,0), const plane3D& plane = plane3D(point3D(0,0,1),vector3D(0,0,1)), vector4D* boundingBox = NULL, TObject** ident = NULL, int lColor = 1, int fColor = 8, int fStyle = 1001) const
      This function is used for drawing a triangle to a root- TCanvas. It will draw a projection of the 3D shape from the eye-point to a plane. The corners of the drawn shape in canvas coordinates are stored in boundingBox. It is drawn using lColor as line color, fColor as fill color and fStyle as fill style. See root-colors and -styles. The parameter ident will return a pointer to the drawn root-primitive to provide the distance-to-primitive functionality.
