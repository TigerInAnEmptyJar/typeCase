Class doc_update_dialog in "doc_update_dialog.h"

back

This widget compares two classes and their methods and variables. If two methods (one from each class) match they are put into the tree-widget and if accepted, the description defined for the "old"-version is copied to the "new" one. Methods, Variables, etc. that don't match are copied to the list-boxes at top for manual assignment.
