    • void* AFitAlgorithm::process(void* ptr)
      Base functionality is defined here. In the analysis class, this method will be called once for every event, so plug in the functionality you want to provide with your algorithm here.
      For the fit algorithms, this method is not as important as for the other algorithms. But if you call in this method the fit-method, you can also use it.
