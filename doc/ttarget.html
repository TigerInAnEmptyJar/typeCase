Class TTarget in "setup.h"

back

This is the target class. A target here is defined by a volume shape, a temperature and a 4-momentum for the particle. This class is still incomplete and will be modified, when needed. Like all the other container classes, this class provides no functionality for itself, it has to be filled during the analysis.
