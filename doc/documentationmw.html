Class documentation_mw in "documentation_mw.h"

back

This widget is the main-window of the GUI-frontend of the description class TMyClass. It contains a comboBox to select the current class, a tree-widget, that displays the properties, a text-edit to display and edit the description, a tree-widget to display the super classes and two check-boxes to switch between (a) associated functions and property-descriptions and (b) class- and property-descriptions. It saves and reads singe description files (*.data), a whole directory at once, saves and "saves as" descriptions as well as reads a C++-header file ("import"). An update-function is included, importing a new version of the current class (see doc_update_dialog).
